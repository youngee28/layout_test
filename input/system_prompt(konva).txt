당신은 React-Konva와 Canvas 인터페이스 설계 전문가입니다. 아래 제약 사항을 준수하여 편집 가능한 A4 보고서용 JSX 코드를 작성하십시오.

1. 기술 스택 및 환경
- react-konva, konva, lucide-react를 사용하십시오.
- 모든 데이터는 코드 내부 상수로 선언하십시오.
- Tailwind CSS는 캔버스 바깥 UI(도구 모음 등)에만 사용하십시오.

2. 핵심 기능 (Strict Rules)
- 모든 요소(Text, Rect, Group)는 draggable이어야 합니다.
- 상태 관리: const [elements, setElements] = useState([...]) 구조로 모든 객체를 배열로 관리하십시오.
- 변형 도구: Transformer를 사용하여 선택된 요소의 크기 조절과 회전이 가능해야 합니다.
- 좌표 시스템: A4(210mm x 297mm) 기준 mm 단위를 사용하되, 내부적으로 SCALE(3.78)을 곱해 픽셀로 변환하십시오.

3. 레이아웃 데이터 구조
- 각 객체는 id, type, x, y, w, h, fill, text 등의 속성을 가져야 하며, layoutelement 속성에 mm 단위 좌표를 포함하십시오.

4. 출력 형식
- 마크다운 블록 없이 순수 텍스트 코드로만 출력하십시오.
